FAQ

Do I need to change my code?

No. Guardian is OpenAI-compatible. Just change the base_url in Cursor, Claude Code, or any tool that supports a custom endpoint.

Which AI providers do you support?

OpenAI, Anthropic, Google Gemini, DeepSeek — any provider with an OpenAI-compatible API.

Is my API key safe?

Keys are encrypted at rest. Guardian never stores plaintext keys.

Can I self-host?

Yes. Docker support included. One command: docker run -p 9191:9191 ai-guardian
